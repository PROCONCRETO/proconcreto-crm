# Ejecución (Capa 3)

Scripts de Python deterministas que el agente ejecuta siguiendo las directivas de `directives/`. **Son herramientas del agente, no del producto** — el producto (lo que corre en el navegador) vive en `js/`, `css/` y `cotizaciones.html`, y no depende de nada de aquí.

Casos de uso típicos: consultas/reportes contra Supabase, procesamiento de los Excel de catálogo, tareas de migración de datos puntuales.

Revisa siempre esta carpeta antes de escribir un script nuevo. Todavía no hay scripts creados.
